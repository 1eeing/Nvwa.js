# Nvwa.js
A javascript interpreter.


## 在线体验
[codesandbox](https://codesandbox.io/s/tender-kirch-ygppj?file=/src/App.js)


## 安装
```bash
npm install nvwajs --save
```


## 如何使用
```js
import createContext from 'nvwajs';

const run = createContext();

run(`console.log('hello world')`);
```
